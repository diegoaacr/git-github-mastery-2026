# Git Commands Guide

Guía simple de los comandos más importantes de Git.

---

## git init

```bash
git init
```

Crea un repositorio nuevo en la carpeta actual.

---

## git clone

```bash
git clone <url>
```

Copia un repositorio remoto a tu computadora.

---

## git status

```bash
git status
```

Muestra el estado actual de los archivos.

---

## git add

```bash
git add .
git add <file>
```

Prepara cambios para el próximo commit.

---

## git commit

```bash
git commit -m "mensaje"
```

Guarda los cambios en el historial del repositorio.

---

## git push

```bash
git push
```

Sube los commits al repositorio remoto.

---

## git pull

```bash
git pull
```

Trae y aplica cambios del repositorio remoto.

---

## git fetch

```bash
git fetch
```

Trae cambios del remoto sin aplicarlos.

---

## git branch

```bash
git branch
git branch <name>
```

Lista ramas o crea una nueva.

---

## git switch

```bash
git switch <branch>
```

Cambia a otra rama.

---

## git merge

```bash
git merge <branch>
```

Une otra rama con la rama actual.

---

## git rebase

```bash
git rebase <branch>
```

Reorganiza el historial de commits.

---

## git reset

```bash
git reset
```

Deshace cambios según el modo que uses.

---

## git revert

```bash
git revert <commit>
```

Revierte un commit sin borrar el historial.

---

## git stash

```bash
git stash
git stash pop
```

Guarda cambios temporales y luego los recupera.

---

## Nota

En la carpeta `docs/` encontrarás la explicación completa de cada comando.
