# Install Git on Windows

Guía para instalar Git en Windows.

---

## 🎯 Objetivo

Instalar Git correctamente en Windows.

---

## ⬇️ Descargar Git

Descarga Git desde su sitio oficial:

👉 https://git-scm.com/download/win

---

## 🧱 Instalación

1. Ejecuta el instalador
2. Sigue los pasos (puedes dejar todo por defecto)

---

## ⚙️ Opciones recomendadas

Durante la instalación:

* Editor: usar el recomendado (Vim o Notepad++)
* PATH: elegir opción recomendada
* HTTPS: usar OpenSSL

---

## ✅ Verificar instalación

Abre **Git Bash** o la terminal y ejecuta:

```bash
git --version
```

---

## ⚠️ Problemas comunes

* Comando no reconocido → reinicia la terminal
* Git no en PATH → reinstalar con opción correcta

---

## 💡 Consejo

Usa Git Bash al inicio si no estás familiarizado con la terminal.
