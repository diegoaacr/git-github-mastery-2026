# Installation

En esta sección vas a instalar Git en tu computadora.

---

## 🎯 Objetivo

Tener Git instalado y listo para usar.

---

## 🧠 Antes de empezar

Git funciona en diferentes sistemas operativos:

* Windows
* macOS

Cada uno tiene su proceso de instalación.

---

## 💻 Elige tu sistema

Sigue la guía según tu sistema:

👉 [Install on Windows](./02-installation-windows.md)
👉 [Install on macOS](./03-installation-mac.md)

---

## ✅ Verificar instalación

Después de instalar Git, abre la terminal y ejecuta:

```bash
git --version
```

Si ves una versión, Git está instalado correctamente.

---

## ⚠️ Problemas comunes

* Git no reconocido → no está en el PATH
* Error de permisos → instalar como administrador

---

## 💡 Consejo

No te compliques con configuraciones avanzadas aún.
Primero asegúrate de que Git funcione.
