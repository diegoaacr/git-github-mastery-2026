# Install Git on macOS

Guía para instalar Git en macOS.

---

## 🎯 Objetivo

Instalar Git correctamente en macOS.

---

## 🧠 Opción 1: Usar Homebrew (recomendado)

Si tienes Homebrew:

```bash
brew install git
```

---

## 🧠 Opción 2: Usar instalador oficial

Descarga desde:

👉 https://git-scm.com/download/mac

---

## 🧠 Opción 3: Xcode Command Line Tools

Puedes instalar Git con:

```bash
xcode-select --install
```

---

## ✅ Verificar instalación

```bash
git --version
```

---

## ⚠️ Problemas comunes

* Git desactualizado → usar Homebrew
* Conflictos de versiones → verificar PATH

---

## 💡 Consejo

Homebrew es la mejor opción para mantener Git actualizado.
