# What is Git

Git es una herramienta que te permite guardar y controlar los cambios de tu código.

---

## 🎯 Objetivo

Entender qué es Git y para qué sirve.

---

## 🧠 Explicación simple

Git guarda el historial de cambios de tu proyecto.

Cada vez que haces un cambio importante, puedes guardarlo como un “punto en el tiempo”.

A eso se le llama **commit**.

---

## 📦 Qué puedes hacer con Git

* Guardar versiones de tu proyecto
* Volver a versiones anteriores
* Trabajar en equipo
* Probar cambios sin romper el código

---

## 🧩 Ejemplo simple

Imagina que tienes un archivo:

```text
app.js
```

Haces un cambio y lo guardas con Git.

Luego haces otro cambio.

Git te permite:

* ver qué cambió
* volver atrás
* comparar versiones

---

## 🧠 Idea clave

Git es como un historial inteligente de tu proyecto.

---

## ⚠️ Importante

Git no es GitHub.

* Git → herramienta
* GitHub → plataforma en la nube

---

## 💡 Por qué es importante

Hoy en día, Git es estándar en la industria.

Si trabajas en desarrollo, lo necesitas.
