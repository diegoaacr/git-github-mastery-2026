# How to use this repository

Este repositorio está diseñado como un curso completo de Git y GitHub.

---

## 🎯 Objetivo

Que puedas aprender desde cero hasta nivel profesional, paso a paso y sin confusión.

---

## 📚 Cómo recorrer el contenido

Sigue siempre el orden de las carpetas:

1. `00-introduction`
2. `01-fundamentals`
3. `02-core-concepts`
4. `03+` (niveles intermedios y avanzados)

👉 No saltes temas. Cada bloque se basa en el anterior.

---

## 🧠 Cómo estudiar

Para cada tema:

1. Lee la explicación
2. Revisa los ejemplos
3. Practica en `exercises/`
4. Haz los `labs/`

---

## 🧪 Practica siempre

No basta con leer.

* Usa la terminal
* Crea repositorios
* Prueba comandos
* Equivócate

Así es como realmente se aprende Git.

---

## 📂 Estructura del repo

* `docs/` → Explicaciones paso a paso
* `examples/` → Ejemplos simples
* `exercises/` → Práctica por nivel
* `labs/` → Ejercicios guiados
* `projects/` → Simulación real

---

## ⚠️ Importante

Git no se aprende memorizando comandos.
Se aprende entendiendo cómo funciona.
