# Prerequisites

Antes de empezar con Git, necesitas algunas cosas básicas.

---

## 🎯 Objetivo

Asegurarte de que tienes todo listo para aprender sin problemas.

---

## 💻 Conocimientos previos

No necesitas saber programación avanzada.

Pero sí es recomendable:

* Saber usar una computadora
* Entender carpetas y archivos
* Saber abrir la terminal

---

## 🖥️ Terminal

Git se usa desde la terminal.

Debes saber:

* Abrir la terminal
* Navegar entre carpetas
* Ejecutar comandos

Ejemplos básicos:

```bash
cd carpeta
ls
pwd
```

---

## 🌐 Cuenta en GitHub

Necesitas una cuenta en GitHub para trabajar con repositorios remotos.

Puedes crearla gratis.

---

## ⚙️ Git instalado

Debes tener Git instalado en tu sistema.

Puedes verificarlo con:

```bash
git --version
```

Si aparece una versión, ya está instalado.

---

## 🧠 Mentalidad

Esto es importante:

* No memorices comandos
* Entiende lo que haces
* Practica constantemente

---

## ⚠️ Importante

Al inicio puede parecer confuso.

Es normal.

Git se entiende mejor con práctica.
