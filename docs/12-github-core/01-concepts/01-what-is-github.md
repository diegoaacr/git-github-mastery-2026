# What is GitHub

GitHub es una plataforma para alojar y colaborar en proyectos Git.

---

## 🎯 Objetivo

Entender qué es GitHub y cuál es su relación con Git.

---

## 🧠 Git vs GitHub

* Git → herramienta de control de versiones
* GitHub → plataforma basada en Git

---

## 🧩 Ejemplo simple

```bash id="gh1"
Git → motor
GitHub → plataforma colaborativa
```

---

## 🎯 Idea clave

Git funciona solo.
GitHub agrega colaboración y herramientas visuales.

---

## 💡 Qué permite GitHub

* alojar repositorios
* colaborar en equipo
* crear Pull Requests
* gestionar issues
* automatizar workflows

---

## 🔄 Flujo típico

```bash id="gh2"
trabajo local
↓
git push
↓
GitHub
↓
Pull Request
↓
review
↓
merge
```

---

## 🧠 GitHub NO reemplaza Git

👉 Git sigue funcionando sin GitHub

---

## 💡 Alternativas a GitHub

* GitLab
* Bitbucket
* Azure DevOps

---

## ⚠️ Importante

GitHub es la plataforma más usada actualmente.

👉 muy importante profesionalmente

---

## 💡 Qué usan las empresas

GitHub normalmente se combina con:

* CI/CD
* code reviews
* branch protection
* releases

---

## ⚠️ Errores comunes

* pensar que Git = GitHub
* depender solo de interfaz gráfica
* no aprender Git primero

---

## 💡 Buenas prácticas

* dominar Git antes de GitHub
* entender el flujo completo
* usar PRs y reviews correctamente
