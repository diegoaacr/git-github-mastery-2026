# Git Cheatsheet

Resumen rápido de los comandos más usados.

---

## 📁 Inicializar repositorio

```bash
git init
git clone <url>
```

---

## 📄 Estado y cambios

```bash
git status
git add .
git add <file>
git diff
```

---

## 💾 Commits

```bash
git commit -m "mensaje"
git commit -am "mensaje"
```

---

## 🌿 Ramas

```bash
git branch
git branch <name>
git switch <branch>
git checkout <branch>
git branch -d <branch>
```

---

## 🔀 Merge y rebase

```bash
git merge <branch>
git rebase <branch>
```

---

## 🌍 Remoto

```bash
git remote -v
git push
git pull
git fetch
```

---

## 📦 Historial

```bash
git log
git log --oneline
git log --graph
```

---

## 🧹 Otros

```bash
git stash
git stash pop
git reset
git revert
```

---
