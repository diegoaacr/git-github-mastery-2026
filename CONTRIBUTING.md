# Contributing

Gracias por querer contribuir.

---

## Cómo contribuir

1. Haz fork del repositorio
2. Crea una rama
3. Realiza cambios
4. Abre un Pull Request

---

## Reglas

* Explicaciones simples
* Ejemplos claros
* No duplicar contenido
* Seguir la estructura del repo

---

## Commits

Usar mensajes claros:

```bash
feat: agregar ejemplo de merge
fix: corregir explicación de rebase
```

---

## Objetivo

Mantener el contenido claro y útil para principiantes.
